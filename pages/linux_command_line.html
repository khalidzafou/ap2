Mémento des commandes Linux

Retrouvez ci-dessous les commandes essentielles vues en atelier SISR. Le tableau précise la description et fournit un exemple de commande à reproduire dans le terminal.

Commande Description Exemple d'utilisation
ls Liste le contenu du répertoire courant. ls -l
cd Change de répertoire. cd /home/utilisateur
pwd Affiche le chemin complet du dossier actuel. pwd
mkdir Crée un nouveau dossier. mkdir projets
rm Supprime un fichier (utiliser avec précaution). rm notes.txt
cp Copie un fichier ou un dossier. cp source.txt sauvegarde/source.txt
mv Déplace ou renomme un fichier. mv rapport.txt archives/rapport-2024.txt

Conseils pratiques

  • Utilisez la touche Tab pour l'auto-complétion et gagner en vitesse.
  • Combinez les commandes avec && pour exécuter des actions à la suite.
  • Consultez les pages de manuel avec man commande pour approfondir vos connaissances.
